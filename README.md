This project was about creating a clone of  Boat-Lifestyle website as the part of construct week project.

## Project
### Work Distribution:
* 1.Home Page by Aniket Bhandarkar and Mukesh Singh
* 2.Login & SignUp page by Mukesh Singh and Tarun Kalra
* 3.Product Page, Offer Zone page and database by Prathamesh Rawool
* 4.Product Detail,Add to Cart and Payment Pop Up  by Hari Ohm Tripathi and Shubham Anand


### Functionalities :
* Slide shows on the landing page.
* Functional navigation bar with user details
* Login signup functionality with the help of local storage.
* Product search from the database in the local storage.
* Product filter functionality by categories.
* Product sort functionality.
* On click product details page.
